# Test

Write or fix tests for a module.
Usage: `/test MODULE_PATH`

---

Use the dev-tester workflow: @dev-tester

Tell it: "Write or fix tests for [MODULE_PATH].
Honour the four critical test contracts. Add fixtures if needed."

Replace `[MODULE_PATH]` with the path provided by the user.

After finishing, remind: run `/review [MODULE_PATH]` next, then `/done 'item name'`.
